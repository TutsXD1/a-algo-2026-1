Dever de Casa - 01
Objetivo
- Comparar empiricamente a eficiência de dois métodos de ordenação e identificar o "ponto de ruptura" onde a teoria assintótica se torna visível na prática.
Exercício
- Implementar o algoritmo Insertion Sort O(n^2)
- Usar a função nativa do Python sorted() que usa o Timsort,(n log n) para fins de comparação.
- Gere listas aleatórias de tamanhos n = [1000, 5000, 10000, 20000, 50000].
- Meça o tempo de execução de ambor os algoritmos para cada n, gerando a saída no terminal de cada um deles usando a biblioteca time.
